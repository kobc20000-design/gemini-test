# Raton AI 블로그 작가 (MVP) - 새 컴퓨터에서 사용하기

## 1. 준비사항 (기존 컴퓨터에서 복사할 것)
- blog 폴더 전체 (특히 .env, main.py, app.py, requirements.txt, run_blog.bat)
- 학습 데이터를 그대로 쓰고 싶다면 chroma_db 폴더도 함께 복사하세요.

## 2. 새 컴퓨터에서 세팅하기
1. Python을 설치하세요 (python.org에서 다운로드, 'Add Python to PATH' 체크 필수!)
2. blog 폴더를 원하는 곳에 복사합니다.
3. 폴더 안에 있는 [run_blog.bat] 파일을 더블 클릭합니다.
4. 자동으로 필요한 라이브러리가 설치되고 웹 브라우저가 열립니다.

## 3. 웹 화면이 열리면
1. 왼쪽 사이드바에 본인의 네이버 블로그 아이디를 입력하세요.
2. [내 최신 글 긁어오기] 클릭 -> [내 말투 학습 (DB 구축)] 클릭.
3. 학습이 완료되면 우측 화면에서 [주제]와 [참고 자료]를 넣고 [생성] 버튼을 누르면 끝!

## 4. 주의사항
- .env 파일 안에 GEMINI_API_KEY가 정확히 들어있는지 확인하세요.
- 인터넷에 연결되어 있어야 합니다.
